Flex布局

flex之前的布局主要方式:

  • normal flow(正常流也叫文档流)
  • float + clear
  • position relative absolute
  • display inline-block
  • 负margin

什么是flex布局:

一种新的布局方式-Flex布局

  1. 块级布局侧重垂直方向 行内布局侧重水平方向，felx布局与方向无关。
  2. flex布局可以实现空间自动分配 自动对齐(felxible：灵活 弹性)
  3. 适用于简单的线性布局 更复杂的布局要交给grid布局。

flex container的属性

属性名 含义
flex-direction 方向
flex-wrap 换行
felx-flow 上面两个的简写
justify-content 主轴方向对齐方式
align-items 侧轴对齐方式
align-content 多行/列内容的对齐方式

基本的代码结构:

flex-direction

flex布局元素按照什么样的方式排,属性值分别有

  • row
  • row-reverse
  • column
  • column-reverse
  • initial
  • inherit

row: 主轴(横向)方向依次排列

row-reverse: 主轴(横向)反方向依次排列

column: 侧轴(竖向)方向依次排列

column: 侧轴(竖向)反方向依次排列

flex-wrap:

如果不指定flex-wrap的值为no-wrap。在默认的情况当我们继续添加元素flex布局会维持正常的排列其实缩小元素的宽度也不会换行。

我们继续添加元素为之前的2倍。之前我们指定的元素的宽度为100px 但是元素的呈现的结果明显小于100px。

但是我们只要把flex-wrap指定为wrap。
这个时候就会维持元素本身的宽度。当父容器的宽度不能维持的话子元素就会换行显示。

justify-content

主轴方向的对齐方向。
一共有7个值分别是:

  • flex-start
  • flex-end
  • center
  • space-beteen
  • space-around
  • inital
  • inherit

首先我们减少容器的元素为3

flex-start：
元素从容器的开头依次排列。

flex-end：
元素从容器的捷尾向前排列。

center：
元素居中排列。

space-between：
元素位于各行之间留有空白的容器内。

space-around：
元素位于各行之前、之间、之后都留有空白的容器内。

align-items
元素侧轴的排列方式分别是

  • stretch 默认值
  • center
  • felx-start
  • felx-end
  • baseline
  • intital
  • inherit

stretch
默认值。元素被拉伸以适应容器。
如果指定侧轴大小的属性值为’auto’，则其值会使项目的边距盒的尺寸尽可能接近所在行的尺寸，但同时会遵照’min/max-width/height’属性的限制。

center:
元素位于容器的中心。弹性盒子元素在该行的侧轴（纵轴）上居中放置。（如果该行的尺寸小于弹性盒子元素的尺寸，则会向两个方向溢出相同的长度）。

felx-start： 元素位于容器的开头。弹性盒子元素的侧轴（纵轴）起始位置的边界紧靠住该行的侧轴起始边界。

felx-end: 元素位于容器的结尾。弹性盒子元素的侧轴（纵轴）起始位置的边界紧靠住该行的侧轴结束边界。
baseline:
元素位于容器的基线上。如弹性盒子元素的行内轴与侧轴为同一条，则该值与’flex-start’等效。其它情况下，该值将参与基线对齐。

flex-item的属性

  • flex-grow 增长比例
  • flex-shrink 收缩比例
  • flex-basis 默认大小
  • flex 上面的缩写
  • order 顺序
  • align-self 自身的对齐方式

初始代码:

flex-grow: 增长比例当空间不够的时候如何分配剩余空间.

元素按照 1:2:3的比例自动分配。

flex-shrink: 在元素空间不够的情况下如何收缩自己。
flex: 是flex-grow flex-shrink flex-basis缩写。

order: 指定元素的排列的顺序。

align-self可以单独设置对齐方式覆盖flex-container设置的值。
